Project Attributions

This project was developed for educational purposes. Below are the resources and tools used in the creation of The Capsule Curator.

Imagery & Inspiration

  • Photography: All fashion and product imagery was curated via Pinterest for aesthetic and educational use.
  • Logo Design: Created using custom SVG paths and Google Fonts.

Technical Tools

  • Image Optimization: All images were compressed and converted to .webp format using the Squoosh App to ensure high performance and fast load times.
  • Typography: 'Playfair Display' and 'Montserrat' fonts provided by Google Fonts.
  • Weather Data: Real-time weather suggestions are powered by the OpenWeatherMap API.
  • Icons: Minimalist icons and symbols sourced from Tabler Icons.
  • Development: Built with HTML5, CSS3, and Vanilla JavaScript.

Validation Tools

  • W3C HTML Validator
  • W3C CSS Validator
  • Google Lighthouse (Performance & Accessibility)
